**Stand:** 19.02.2026

## Welche Daten werden verarbeitet?

SnapHunt speichert alle Daten **ausschließlich lokal auf deinem Gerät**.
Es findet keine Übertragung an eigene Server statt.

### Lokal gespeicherte Daten
- Fotos, die du im Rahmen von Aufträgen aufnimmst
- Auftrags-Metadaten (Titel, Beschreibung, Status, Zeitstempel)
- Dein OpenAI API-Schlüssel (verschlüsselt im iOS Keychain / Android Keystore)

### GPS-Standort (optional)
Wenn du "Standortbasierte Aufträge" nutzt, wird dein ungefährer Standort
an OpenAI (OpenAI, L.L.C., San Francisco, USA) übertragen, um ortsspezifische
Aufgaben zu generieren. Diese Daten unterliegen der Datenschutzerklärung von
OpenAI: https://openai.com/policies/privacy-policy

### EXIF-Daten
Alle aufgenommenen Fotos werden automatisch von GPS-Koordinaten und
Geräteinformationen (EXIF-Daten) bereinigt, bevor sie gespeichert werden.

## Datenlöschung
Sämtliche lokalen Daten werden gelöscht, wenn du die App deinstallierst.
Es gibt keine Möglichkeit, Daten über mehrere Geräte hinweg zu synchronisieren
oder nach einer Deinstallation wiederherzustellen.

## Rechte nach DSGVO
Da alle Daten lokal gespeichert werden, kannst du sie jederzeit durch
Deinstallation der App vollständig löschen.
